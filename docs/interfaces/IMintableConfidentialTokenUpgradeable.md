# Solidity API

## IMintableConfidentialTokenUpgradeable

Initializer interface for upgradeable mintable confidential token deployments.

### initialize

Initializes the contract for proxy deployment.

```solidity
function initialize(string name, string symbol, string version_, address initialAuthority) external
```

#### Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| name | string | Name of the token. |
| symbol | string | Symbol of the token. |
| version_ | string | Version of the token. |
| initialAuthority | address | Initial authority address. |
